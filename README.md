# Poorni's Pages — A Book Blog

A cozy book blog built with HTML & CSS, featuring reviews, reading lists, and literary musings.

## About

Poorni's Pages is a personal book blog celebrating stories, authors, and the joy of reading. The site features a warm, elegant design with sage greens, cream tones, and gold accents.

## Features

- Book reviews and ratings
- Reading lists and recommendations
- Spoonies-friendly content

## Tech

Plain HTML & CSS — no frameworks, no build step. Just open `index.html` in a browser.
